# 🚀 SAE15 Razi, Malik, Hugo, Zidane, AbdoulHakim

Ce projet génère des informations synthétiques à partir d'un dataset portant sur les expérimentations menées actuellement par l'ARCEP en France sur l'utilisation de la 5G.
Il permet de générer sous forme de graphique et d'une carte intéractive de synthétiser les informations.
Ce projet réalisé en groupe, composé de :
Zidane, Malik, AbdoulHakim, Hugo et Razi.

---

## ✨ Livrable générés

- **Diagramme en camembert de l'utilisation des fréquences**  
- **Diagramme en barres des régions les plus représentés** 
- **Diagramme en barres des expérimentateurs les plus représentés**
- **Carte interactive grâce à l'outil Folium**

---

## 🛠 Téléchargement

1. Clone le dépôt GitHub:
   ```bash
   git clone https://github.com/RRazi44-bis/SAE15.git
   cd SAE15
   pip install -r requirements.txt
   ```

2. Lancer le projet grâce à:
   ```bash 
   python main.py
   ```
   Les graphiques seront générés dans le même répertoire et une fenêtre tkinter s'ouvrira.

   
---


## 📌 Utilisation

Une fois téléchargé et lancé, une fenêtre apparaîtra où pour ouvrir chaque graphique et la carte il suffira de cliquer sur les boutons correspondants.
